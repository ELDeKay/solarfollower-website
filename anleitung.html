So stellen Sie ihr System ein

Inbetriebnahme des Mikrocontrollers

  • Öffnen Sie die Website 192.169.1.42 in einem neuen Tab im Browser
  • Verbinden Sie sich mit dem WLAN des Mikrocontroller
  • Die Seite zeigt nun Eingabefelder an, in denen Sie den Namen des gwünschten WLAN-Netzwerkes und dessen Kennwort eingeben sollen
  • Sollte Sie keine Website aufrufen können, überprüfen Sie ob Sie mit dem WLAN des Mikrocontrollers verbunden sind
  • Eine Nachricht auf der Website 192.168.1.42 zeigt an ob dieser Schritt gelungen ist

Funktionen der Website

Hauptmenü

  • Durch Anklicken der Kacheln im Hauptmenü gelangt man zum gewünschten Untermenü.
  • Klicken Sie zunächst im Hauptmenü die Kachel "Einstellungen"

Einstellungen

  • Downloaden Sie nun die "Shelly Smart Control" App aus dem PlayStore
  • Richten Sie ihren Shelly ein und lesen Sie die IP-Adresse, des nun im WLAN-Netzwerk integrierten Shelly, aus
  • Tragen Sie nun ihren Breiten- und Längengrad des Standortes ihres "mysolarfollower" ein
  • Tragen Sie die IP-Adresse des Shelly in das passende Eingabefeld ein
  • Klicken Sie nun auf die Buttons um die jeweiligen Daten an den Mikrocontroller zu schicken
  • Durch das Klicken des "zurück zum Hauptmenü" Buttons gelangen Sie zurück ins Hauptmenü

Aktuelle Wetterdaten

  • In diesem Menü sehen Sie alle Wetter-abhängigen Daten
  • Unten können Sie, falls erwünscht Ihre WLAN-Steckdose einrichten, sodass Sie optimal genutzt wird
  • Tragen Sie die Anzahl der Stunden ein (als ganze Zahl), wie lange Ihre WLAN-Steckdose in Betrieb sein soll
  • Der Mikrocontroller berechnet den optimalen Start für das Einschalten der Steckdose
  • Tragen Sie dafür die IP-Adresse ihrer WLAN-fähigen-Steckdose, sowie die UTC-Abweichung ihrer Ortszeit ein
  • Klicken Sie auf den Button "Sende WLAN Steckose Einstellungen"

Aktuelle Paneeldaten

  • HIer sehen Sie alle Paneel-abhängigen Daten
  • Der Button "Kalibrierung stellt das Gestell ein, sodass es in eine Endstellung verfährt und dann sich automatisch mittig hinstellt
  • Nun steht das Gestell in Richtung (des berechneten) Süden und muss dementsprechend ausgerichtet werden
